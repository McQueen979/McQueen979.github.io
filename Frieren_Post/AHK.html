AHK入门介绍

一、AHK 核心认知（新手必懂）

1.1 什么是 AHK？

AutoHotkey（简称AHK）是Windows免费开源自动化工具，核心作用：自定义键盘/鼠标快捷键，替代所有重复手动操作。

无需复杂编程，仅简单语法，即可实现：

  • 一键填充固定文本（手机号、AI提示词、地址、话术）

  • 一键模拟系统按键（复制、粘贴、窗口切换）

  • 一键打开网页、文件夹、软件

  • 自定义闲置按键，解放常用操作

1.2 核心运行逻辑

  • 脚本文件后缀：.ahk

  • 运行后后台常驻，右下角托盘显示绿色H图标

  • 修改代码后必须 Reload This Script（重新加载） 才会生效

二、从零搭建 AHK 环境（完整步骤）

2.1 前期准备

已下载安装 AutoHotkey，默认安装即可，无需额外配置。

2.2 新建专属脚本文件

  1. 桌面空白处 右键 → 新建 → AutoHotkey Script

  2. 重命名为：我的快捷工具.ahk（后缀必须是.ahk，不可修改）

  3. 右键脚本 → Edit Script（编辑脚本），打开记事本编辑

  4. 删除所有系统默认代码，粘贴自定义代码

  5. Ctrl+S 保存，关闭记事本

  6. 双击脚本运行，托盘出现绿色H即运行成功

2.3 永久开机自启（必备）

  1. 按下 Win+R，输入 shell:startup 回车

  2. 打开开机启动文件夹

  3. 将桌面的 我的快捷工具.ahk 复制粘贴到该文件夹

效果：后续开机自动后台运行脚本，无需手动打开。

三、AHK 核心基础语法（新手必背）

3.1 快捷键绑定固定格式

所有快捷键统一标准写法（规范、零报错、好维护）

关键规则：

  • 多行命令：必须加 return，否则代码串行出错

  • 单行命令：可省略 return，但新手统一全部带return，规避bug

3.2 AHK 专属符号对应表（万能通用）

  • ^ = Ctrl 键

  • ! = Alt 键

  • + = Shift 键

  • # = Win 键

3.3 常用按键代码对照

  • F1-F12：直接写 F1::F12::

  • 小键盘数字：Numpad1-Numpad9

  • 鼠标侧键：XButton1（前进）、XButton2（后退）

四、四大核心指令（你100%会用到）

AHK指令成千上万，仅需掌握以下4类，满足99%日常需求

4.1 SendInput 指令（文本填充专用）

作用：向输入框填充自定义文字（手机号、提示词、话术、网址）

使用前提：必须鼠标点击输入框，光标激活生效

优势：不乱码、不冲突、输入稳定

4.2 Run 指令（打开/跳转专用）

作用：无需输入框，直接执行跳转/打开

可打开内容：网页链接、本地文件夹、软件、系统工具

和SendInput核心区别：Send是“填文字”，Run是“跑程序/开页面”

4.3 Send 指令（模拟系统按键）

作用：模拟键盘组合键（复制、粘贴、切换窗口等系统操作）

4.4 ToolTip 指令（悬浮提示）

作用：按键后屏幕弹出短暂提示，确认操作成功

五、按键取舍指南（新手最优按键方案）

5.1 绝对不要修改的按键（系统核心功能）

  • 单字母 a-z、数字0-9（打字必备）

  • Ctrl+C/V/Z/S/A（复制、粘贴、撤销、保存、全选）

  • 回车、空格、Tab、Shift、Win基础组合键

5.2 最优闲置按键（无冲突、最适合自定义）

首选：F7 F8 F9 F10（系统默认功能极少，几乎零冲突）

次选：Alt+1~Alt+9（手感极佳，无系统占用）

备选：小键盘全部按键（纯闲置，适合批量快捷指令）

六、超高价值拓展：可一键填充的所有场景

整理所有适配AHK的自定义填充内容，直接复用即可

6.1 个人信息类（填表、登录、注册）

手机号、邮箱、姓名、收货地址、邮编，告别手动输入

6.2 AI提示词类（GPT/豆包专属）

一键填充：翻译、详细解释、总结、润色、分点解答、代码排错

6.3 办公话术类（微信/钉钉/邮件）

工作回复、请假、汇报、邮件首尾固定话术

6.4 资源快速打开类

常用网页、本地文件夹、工作目录、学习网站（新概念英语网站等）

6.5 快捷操作类

一键复制、一键粘贴、一键切换窗口，替代鼠标操作

七、你专属的最终成品脚本（直接即用）

整合你所有需求，可直接复制替换全部代码，修改个人信息即可使用

八、常见问题排查（100%解决新手报错）

8.1 按键没反应

  • 右键托盘绿色H图标 → Reload This Script 重新加载

  • 检查小键盘NumLock是否亮起（小键盘按键专用）

  • 检查代码是否缺少return、标点错误

8.2 文本乱码

固定使用 SendInput {Text}内容 格式，杜绝乱码

8.3 功能混淆问题

  • 需要填文字 → 用 SendInput

  • 需要打开/跳转 → 用 Run

  • 需要模拟按键 → 用 Send

8.4 关闭/暂停脚本

  • 临时暂停：托盘H右键 → Suspend Hotkeys

  • 彻底关闭：托盘H右键 → Exit

九、新手进阶拓展方向

  1. 新增更多AI提示词、办公话术、个人模板

  2. 设置鼠标侧键快捷操作，解放键盘

  3. 添加Tooltip悬浮提示，操作可视化

  4. 配置窗口置顶、最小化、定时提醒等实用功能
